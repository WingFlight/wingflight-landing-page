Usage Guidelines

  • Primary Red (#d40000) - Use for main CTAs, logo, primary interactions
  • Dark Gray (#4e4e4e) - Use for body text, headlines, primary content
  • Light Gray (#f5f5f5) - Use for backgrounds, section dividers
  • Dark Red (#a40000) - Use for hover states and emphasis
  • Light Red (#ff4d4d) - Use for alerts and secondary highlights
  • Maintain contrast - Ensure WCAG AA compliance (4.5:1 for text)
